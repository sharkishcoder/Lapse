## Day 1 — Learning the basics before touching code
**Date:** 

**What I did:**
- Talked through project idea and picked a rough plan
- Learned what a "tech stack" is

**New vocab:**
- **Stack**: the set of tools used to build an app, layered by job —
  frontend (what you see), backend (logic behind the scenes),
  database (permanent info storage), file storage (for big files
  like videos), and hosting (where the site lives online).
- **PWA (Progressive Web App)**: a website that can be "installed"
  to a phone home screen and feels like a native app, without
  needing the App Store.
- **Devlog**: a running journal of what you built and learned,
  session by session.

**What I learned:**

## Glossary (running list — add new terms here as you learn them)

- **Stack**: the tools used to build an app, layered by job.
- **Frontend**: the part of the app the user sees and interacts with.
- **Backend**: the behind-the-scenes logic the frontend talks to.
- **Database**: where structured info (like "5-day streak") is stored.
- **Storage**: where files (like videos) are stored, separate from
  the database.
- **PWA**: an installable website that feels like a native app.
- **Devlog**: a session-by-session log of what you built and learned.
